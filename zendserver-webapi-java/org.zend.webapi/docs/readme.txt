
=========================================
Zend WebApi Client for Java
=========================================

1. Install
-------------
To install Zend WebApi client for Java you'll need to extract the SDK into a 
directory.

The following resources are available:

	/src          - source code 
	/test         - test code
	/examples     - code examples 
	/lib          - compiled library including 3rd part directory 
	/docs         - API JavaDoc resources 
	/junit        - the Latest JUnit reports generated on this compile time
	copyright.txt - copyright note
	license.html  - license specification
	readme.txt    - this file
